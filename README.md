## 概述

本场分享会的主题是「从会用AI，到用好AI」，核心内容围绕 **四大工程体系**：

1. **提示词工程（Prompt Engineering）** — 教你怎么跟 AI 说话
2. **上下文工程（Context Engineering）** — 让 AI 知道它在跟谁打交道
3. **驾驭工程（Harness Engineering）** — 给 AI 套上缰绳
4. **循环工程（Loop Engineering）** — 做→查→修→再输出，直到满意
