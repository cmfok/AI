## 三、交互与功能需求

### 3.1 页面切换

| 属性 | 值 |
|------|-----|
| 翻页方式 | 控制台驱动（POST `/api/control/go` {slide: N}） |
| 动画效果 | **方案一：左右翻页 + 渐变全黑过渡** |
| 过渡流程 | 当前页渐隐 350ms → 全黑 → 切换内容 → 渐亮 300ms |
| 键盘控制 | ← → ↑ ↓ Space（展示页自身） |
| 页面指示 | 底部居中 "N / 19"，monospace 数字，半透明毛玻璃背景 |
| URL 参数 | `?go=N` 直接跳转到指定页 |

### 3.2 观众问答（Q&A）

| 维度 | 设定 |
|------|------|
| 数据源 | 控制台通过 `/api/control/qa-show`（POST）推送 Q&A 到展示页 |
| 轮询 | `/api/control/qa-overlay`（GET）+ `/api/questions`（GET） |
| 桌面端 | **右侧固定面板**（340px 宽），半透明毛玻璃黑底，可收起/展开 |
| 手机端 | **底部抽屉**（Bottom Sheet），收起时细栏"💬 N"，展开半屏 |
| 切换按钮 | 右侧浮动按钮，显示未读数角标（橙色 #ff6b35） |

### 3.3 控制台功能

| 功能 | 说明 |
|------|------|
| 上下翻页 | ← 上一页 / 下一页 → 按钮 |
| 页面网格 | 自动填充 19 个 P1~P19 方格，点击跳转 |
| 演讲稿面板 | 📝 Tab：显示当前页脚本内容，支持 markdown 渲染 |
| 观众问题面板 | 💬 Tab：显示待处理提问（语音监听）+ 已接收扫码问题 |
| AI 问答面板 | 🎤 Tab：语音识别（浏览器 SpeechRecognition API）+ 文字兜底 |
| 同步状态 | 底部显示与展示页的同步连接状态 |

### 3.4 演讲稿渲染规范

| 标记 | 渲染效果 | 示例 |
|------|---------|------|
| `**文本**` | 加粗 `<strong>` | `**核心观点**` → **核心观点** |
| `*文本*` | 斜体 `<em>` | `*备注*` → *备注* |
| `` `文本` `` | 代码 `<code>` | `` `go: N` `` → `go: N` |
| ` ``` ` 代码块 | 预格式 `<pre><code>` | —— |
| `\n` | 换行 `<br>` | 段落分隔 |
| `💡` / `⚠` / `🔮` | Emoji 保留 | 标识洞察/警告/未来 |

### 3.5 问答分类引擎

- 基于 TF-IDF + 余弦相似度（纯 CPU 零依赖）
- 每页有 `title` + `tags` 用于分类匹配
- 封面页/过渡页（`COVER_PAGES` = {1, 3, 6}）不参与分类
- 无法归类的问题默认落到第 19 页

## 四、后端架构

### 4.1 路由表

| 路径 | 方法 | 说明 | 文件 |
|------|------|------|------|
| `/tech` | GET | 新展示页（19页+渐变过渡+Q&A） | `tech_display.html` |
| `/tech-control` | GET | 新控制台 | `tech_control.html` |
| `/api/control/go` | POST | 翻页 {slide: N} | `app_current.py` |
| `/api/control/current` | GET | 获取当前页 {slide: N} | `app_current.py` |
| `/api/control/qa-show` | POST | 推送 Q&A 到展示页 | `app_current.py` |
| `/api/control/qa-overlay` | GET | 轮询当前 Q&A | `app_current.py` |
| `/api/qa/ask` | POST | AI 问答匹配 | `app_current.py` |
| `/api/questions` | GET/POST | 扫码提问 | `app_current.py` |

### 4.2 数据流

```
控制台 ──POST /api/control/go──→ 展示页（轮询 /api/control/current → 渐变过渡）
控制台 ──POST /api/control/qa-show──→ 展示页（轮询 /api/control/qa-overlay → 右侧面板）
观众 ──POST /api/questions──→ 控制台（轮询 /api/questions → 问题列表）
语音 ──POST /api/qa/ask──→ 匹配 QA 知识库 → 翻到对应页 + 推送 Q&A
```

### 4.3 QA 知识库

- 文件：`qa_knowledge.json`
- 结构：`{qa_pairs: [{id, question, answer, slide, keywords}, ...]}`
- 当前预配 19 条问答，覆盖所有幻灯片主题
