# AI 真垃圾 · PPT 大纲

## 设计系统

> **风格**：瑞士国际主义 × 苹果极简
> - **底色交替**：大章节黑底，内容页白底
> - **主色调**：暖白 `#FAFAF8` / 近黑 `#0A0A0A`
> - **点缀色**：浅蓝 `#4A90D9` / 安全橙 `#FF6B35`（仅关键数据）
> - **字体**：Inter（英）+ Noto Sans SC / 微软雅黑（中），全程无衬线
> - **核心规则**：直角纯色、无圆角无阴影、图标放蓝色圆圈内
> - **标题**：封面居中，内容页左上，超大加粗

## 排版铁律（强制）

### 一、案例对比页（Slide 9/11/14/16）

每个落地案例页统一采用 **左右对比结构**，但呈现方式分两种：

**实时演示型（Slide 9/11）** — 嵌入两个并排对话框区域，现场复制Prompt、实时跑AI
**截屏/流程图型（Slide 14/16）** — 左❌右✅用提前准备的截图或流程图

```
实时演示页：                        截屏/流程图页：
┌──────────┬──────────┐          ┌──────────┬──────────┐
│ 对话框截图  │ 对话框截图  │          │ 人工/单次  │ 自动化/循环 │
│ （现场填充） │ （现场填充） │          │ （截图）   │ （截图）   │
└──────────┴──────────┘          └──────────┴──────────┘
  关键变量：____                    关键变量：____
```

- 左栏用橙色系表达警示，右栏用蓝色系表达达成
- 底部一行点明"关键变量"
- 翻页后接"揭晓工程原理"页
- 提示词/演示素材从演讲稿对应段落提取

| Slide | 案例 | 呈现方式 | 备注 |
|:---:|------|:---:|------|
| 9 | 语音→PPT | 提前准备好 | 传统AI做的PPT vs 本场分享会PPT（元演示） |
| 11 | 约会翻车 | 实时演示 | 双对话框：无背景信息 vs 有女孩资料 |
| 14 | 运费对账+招聘 | 流程图+实时演示 | 运费对账用流程图；招聘简历用 demo_analyzer.py 双栏对比 |
| 16 | 生日营销 | 流程图 | PPT展示循环执行流程，口述逻辑 |

### 二、技术演进页（Slide 17 循环工程）

采用 **三栏结构**：

```
┌──────────────┬──────────────────┬──────────────────┐
│ 循环流程       │ ⚠ 现在：痛点       │ 🔮 未来：方向       │
│ 当前怎么做     │ 卡在哪            │ 往哪走            │
│ （蓝）        │ （橙）            │ （蓝+橙过渡）      │
└──────────────┴──────────────────┴──────────────────┘
```

- 形成「现状 → 痛点 → 方向」认知闭环
- 右栏"未来设想"：多Agent并行 + 实时通讯对齐，对比当前单Agent串行瓶颈

### 七、工程揭晓页（Slide 10/12/15/17）— 四部曲模板

四个工程揭晓页共用统一模板，参考 `AI赋能-滚动展示页.html` 同源布局：

```
CHAPTER XX · ENGINEERING NAME
工程中文名
一句话核心观点
┌────────────────────────────┬──────────────────────────┐
│ 📖 故事卡片                 │ 规则/步骤卡片              │
│                            │                          │
│ ❌ 反面案例                 │ RULE/STEP/LEVEL 01       │
│ → 后果                    │ 标题 + 一句话含义           │
│                            │                          │
│ ✅ 正面案例（有了工程后）      │ RULE/STEP/LEVEL 02       │
│ → 效果                    │ 标题 + 一句话含义           │
│                            │                          │
│                            │ RULE/STEP/LEVEL 03       │
│ 💡 关键洞察                 │ 标题 + 一句话含义           │
│                            │                          │
│                            │ RULE/STEP/LEVEL 04       │
│                            │ 标题 + 一句话含义           │
└────────────────────────────┴──────────────────────────┘
```

- **底色**：深蓝渐变黑底，每章微调渐变角度/色相区分（`#0a0a1a → #111 → #0d0d0d` 系列）
- **左上**：CHAPTER 标签（蓝色小胶囊）→ 大字标题（含英文副标题）→ 一句描述
- **左栏**：story-card（半透明浅黑直角卡片），含 📖 emoji、❌/✅ 对比、💡 insight 条
- **右栏**：规则/步骤卡片（浅黑半透明直角小块），编号 + 标题 + 一句话含义
- **底部**：蓝色高亮 insight 条 + 灰色工具标签行
- **Slide 17 特殊处理**：上半四部曲模板 + 下半/翻页三栏展开（循环流程·痛点·未来）
- 四页视觉连续统一，仅渐变底色和章节编号切换，形成系列感
