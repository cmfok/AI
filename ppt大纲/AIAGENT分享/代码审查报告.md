## 🔴 严重问题（3项）

### 1. `index.html` — autoScroll 双定时器冲突

**位置**：第 800-809 行 与 第 822-826 行  
**文件**：`ppt大纲/AIAGENT分享/index.html`

同一个 `d.autoScroll` 字段被**两个代码块同时处理**：

- **块1**（第 800-809 行）：使用 `window._autoTimer`
- **块2**（第 822-826 行）：使用本地 `_autoTimer` + `_lkAuto` 去重变量

两个块各自创建 `setInterval` 滚动同一个弹窗，导致**两个定时器同时运行**，滚动速度翻倍且行为混乱。

**修改建议**：删除其中一套处理逻辑。建议保留块2（第 822-826 行），删除块1（第 800-809 行），因为块2有完整的 `_lkAuto` 去重机制和 `startSmoothScroll`/`stopSmoothScroll` 封装。

```javascript
// ❌ 删除以下重复代码（第800-809行）
if(d.autoScroll!==undefined){
  if(d.autoScroll && !window._autoTimer){
    var pp=document.getElementById("mdPopup");
    if(pp) window._autoTimer = setInterval(function(){ pp.scrollTop += 12; }, 12);
  }
  if(!d.autoScroll && window._autoTimer){
    clearInterval(window._autoTimer);
    window._autoTimer = null;
  }
}

// ✅ 保留以下代码（第822-826行）
if(d.autoScroll!==undefined && d.autoScroll!==_lkAuto){
  _lkAuto=d.autoScroll;
  if(d.autoScroll){ startSmoothScroll(); }
  else { stopSmoothScroll(); }
}
```

### 7. 图片/iframe 资源路径依赖反向代理

**文件**：`index.html` 多处

```html
<img src="/ai/icon/deepseek.svg">
<iframe src="/ai/excel_animation.html">
```

Flask 配置 `static_url_path=''`，静态文件在根路径。带 `/ai/` 前缀的路径必须依赖 Nginx 等反向代理规则。

**修改建议**：在项目文档中明确标注反向代理的路径映射要求（如 `location /ai/ { proxy_pass http://localhost:8008/; }`），或调整 Flask 路由统一处理 `/ai/` 前缀。
