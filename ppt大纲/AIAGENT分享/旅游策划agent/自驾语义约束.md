# 自驾语义约束 — "自驾"一词的两种含义

> ⚠️ **核心问题**：人类说「自驾」时，通常不区分两种含义，但 Agent 必须区分，否则费用估算完全错误。

---

## 一、两种自驾的定义

| 类型 | 人类说法 | Agent 关键字 | 定义 | 费用估算方式 |
|------|---------|-------------|------|------------|
| **全程自驾** | "自己开车去"、"全程自驾"、"开车过去" | `全程自驾`、`自己开车去`、`开车过去`、`自驾过去` | 从家开自己的车，跨城到目的地 | 距离 × 油费 + 过路费（如广州→大理 1600km ≈ ¥1,920） |
| **当地租车** | "租车自驾"、"到了租车"、"落地租车"、"当地自驾" | `当地租车`、`租车自驾`、`到了租车`、`落地租车` | 公交到目的地后，在当地租车 | 日租金(¥200~400) × 天数 + 保险(¥50/天) + 当地油费(¥80/天) |

---

## 二、歧义处理规则

### 规则 1：用户只说「自驾」，不明确类型

```
用户输入: "大理5天 自驾"
正确处理: 默认按「当地租车」处理，并输出警告
错误处理: 直接按「全程自驾」算 1600km 油费
```

### 规则 2：用户同时提到两种

```
用户输入: "全程自驾和当地租车哪个合适"
正确处理: drive_mode = "both"，输出对比表
```

### 规则 3：亲子游场景的特殊性

| 场景 | 推荐方式 | 原因 |
|------|---------|------|
| 距离 > 500km | **当地租车** | 长途孩子坐不住，晕车风险高 |
| 距离 < 500km | 全程自驾 | 行李方便，成本可控 |
| 3岁以下孩子 | **当地租车** | 飞机/高铁更快，到当地灵活自驾 |
| 5天以内 | **当地租车** | 节省往返交通时间 |
| 7天以上 | 全程自驾 | 时间充裕，成本更低 |

---

## 三、当地租车费用估算公式

```python
# 经济型 SUV（如丰田RAV4/本田CR-V）
日租金   = 200 ~ 400 元/天 （取 ¥300 中间值）
保险     = 50 元/天
当地油费 = 80 元/天 （按每天行驶 60km 估算）

总费用 = (日租金 + 保险 + 当地油费) × 天数
       = (300 + 50 + 80) × 天数
       = 430 × 天数

示例：
  5天 → ¥2,150
  7天 → ¥3,010
  3天 → ¥1,290
```

## 四、输出方案中的自驾呈现

### 当 drive_mode = "both" 时：

方案中输出对比表格：

| 维度 | 全程自驾 🚗 | 当地租车 🏎️ |
|------|-----------|----------|
| 交通方式 | 广州→大理 1600km | 飞机 + 大理租车 |
| 单程耗时 | 16h | 2.5h 航班 |
| 交通费用 | ¥1,920 (油+过路费) | ¥3,600 机票 + ¥2,150 租车 |
| 亲子友好度 | ⭐⭐ | ⭐⭐⭐⭐ |
| 适合场景 | 时间充裕 | 5天以内短途 |

### 当 drive_mode = "当地租车" 时：

方案中只输出当地租车明细：

| 费用项 | 金额 |
|--------|------|
| 日租金 | ¥300/天 |
| 保险 | ¥50/天 |
| 当地油费 | ¥80/天 |
| **合计** | **¥2,150（5天）** |

## 五、与 Loop 自查的联动

在 [loop.md](loop.md) 的自查环节中增加 **自驾语义校验**：

```
🔍 自驾语义校验:
   ├── 用户输入是否包含"自驾/开车" → 是
   ├── 是否明确区分"全程自驾" vs "当地租车" → 否
   ├── 处理方式 → 默认「当地租车」+ 输出警告
   └── ✅ 已正确区分 | ⚠️ 存在歧义
```
