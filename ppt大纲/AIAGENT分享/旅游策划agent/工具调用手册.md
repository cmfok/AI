## 二、工具详细说明

### 2.1 酒店搜索

```bash
npx --yes @fly-ai/flyai-cli search-hotel \
  --dest-name "{目的地}" \
  --check-in-date {YYYY-MM-DD}
```

**参数说明**：

| 参数 | 必填 | 说明 | 示例 |
|------|------|------|------|
| `--dest-name` | ✅ | 目的地城市名 | `"大理"` |
| `--check-in-date` | ✅ | 入住日期 | `"2026-07-12"` |

**返回数据**：
```json
{
  "data": {
    "itemList": [
      {
        "name": "酒店名",
        "price": "¥100",
        "star": "舒适型",
        "interestsPoi": "近大理古城"
      }
    ]
  }
}
```

**降级策略**：
- 主策略失败 → 改用 `keyword-search --query "{目的地}亲子酒店双床房推荐"`
- 降级仍失败 → 标注 "⚠️ FlyAI 无数据，建议飞猪APP查询"

## 三、调用规则

### 3.1 超时控制

所有 FlyAI 调用默认超时 **25秒**。超时视为失败，触发降级。

### 3.2 重试策略

| 场景 | 策略 |
|------|------|
| 超时 | 不重试，直接降级 |
| JSON解析失败 | 不重试，记录错误 |
| 退出码非0 | 不重试，记录错误并降级 |

### 3.3 降级链

```
专项搜索 → keyword-search → 人工提示（⚠️）
```

每条失败路径都有清晰的降级方案，确保 Agent 不会「卡死」。

## 四、🚗 自驾费用估算（内置工具，不依赖 FlyAI）

### 4.1 全程自驾估算

Agent 内置城市间距离映射表，按以下公式计算：

```python
# 距离查询（dist_map）
dist = dist_map.get(f'{出发地}-{目的地}', 800)  # km

# 费用计算
油费   = 距离 × 0.7   # 元/km
过路费 = 距离 × 0.5   # 元
合计   = 距离 × 1.2   # 油+过路费
```

示例：广州→大理 1600km → ¥1,920

### 4.2 当地租车估算

> ⚠️ **关键语义区分**：FlyAI 不提供租车搜索，Agent 使用内置公式估算。

```python
日租金   = 300     # 经济型 SUV，¥200~400/天取中间值
保险     = 50      # 元/天
当地油费 = 80      # 元/天（按每天行驶 60km 估算）

总费用 = (日租金 + 保险 + 当地油费) × 天数
       = (300 + 50 + 80) × 天数
       = 430 × 天数
```

示例：5天 → ¥2,150 | 7天 → ¥3,010

### 4.3 语义区分触发条件

| 用户输入 | drive_mode | 估算内容 |
|----------|-----------|---------|
| "全程自驾" / "开车过去" | `全程自驾` | 仅全程自驾距离/费用 |
| "租车自驾" / "当地租车" / "到了租车" | `当地租车` | 仅当地租车费用 |
| "自驾"（歧义） | `当地租车` + ⚠️ | 当地租车 + 歧义警告 |
| "全程自驾和当地租车哪个好" | `both` | 双栏对比表 |

详见 [自驾语义约束.md](自驾语义约束.md)
