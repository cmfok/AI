## 三、调用方式

### 3.1 在 Agent 中调用

```python
from data_fetcher import query, calc_drive_cost, calc_rental_cost

# 酒店查询
r = query('search-hotel --dest-name "大理" --check-in-date 2026-07-12')
hotels = r["data"]["itemList"]  # 格式与 flyai 完全兼容

# 全程自驾估算
drive = calc_drive_cost("广州", "大理")
# → {"km": 1600, "h": 16.0, "gas": "¥1120", "toll": "¥800", "total": 1920}

# 当地租车估算
rental = calc_rental_cost(5)  # 5天
# → {"per_day": 300, "days": 5, "insurance": 250, "gas_local": 400, "total": 2150}
```

### 3.2 完整运行

```bash
cd 旅游策划agent/
python agent_v5.py "一家三口大理5天4夜 预算2万 从广州出发 飞机 当地租车自驾 亲子"
```

## 四、自驾语义区分

| 用户输入 | drive_mode | 估算内容 |
|----------|-----------|---------|
| "全程自驾" / "开车过去" | `全程自驾` | 仅全程自驾距离/费用 |
| "租车自驾" / "当地租车" / "到了租车" | `当地租车` | 仅当地租车费用 |
| "自驾"（歧义） | `当地租车` + ⚠️ | 当地租车 + 歧义警告 |
| "全程自驾和当地租车哪个好" | `both` | 双栏对比表 |

详见 [自驾语义约束.md](自驾语义约束.md)

## 五、扩展知识库

如需增加新城市，编辑 `data_fetcher.py`：

```python
# 在 HOTELS 字典中新增
"新城市": [
    {"name": "酒店名", "price": "¥299", "star": "舒适型",
     "interestsPoi": "近商圈", "type": "酒店"},
    ...
]

# 在 POIS 字典中新增
"新城市": [
    {"name": "景点名", "type": "自然风光", "ticket": "¥50",
     "kid_friendly": True, "desc": "描述"},
    ...
]

# 在 TRAIN_ROUTES 中新增火车路线
"广州-新城市": [
    {"_name": "车次 广州-新城市 XXh", "_price": "¥xxx"},
    ...
]
```
