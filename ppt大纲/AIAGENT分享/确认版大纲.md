# 技术规范

## 一、技术架构

```
控制台（control.html）                  展示页（index.html）
  └─ POST /api/control/go                  └─ 轮询 GET /api/control/current
       {slide: N}                                → 渲染对应 Slide
  └─ POST /api/control/qa-show             └─ 轮询 GET /api/control/qa-overlay
  └─ POST /api/qa/ask                      └─ 轮询 GET /api/questions
  └─ GET /api/questions
```

## 二、语音问答功能

| 功能 | 说明 |
|:-----|------|
| 触发 | 演讲者在手机控制台点击按钮启动语音识别 |
| 识别 | 浏览器 SpeechRecognition API → 语音转文字 |
| 回传 | 文字实时回传控制台问题面板 |
| 匹配 | AI 自动识别问题对应哪一页 PPT → 自动跳转 |
| 回复 | 页面显示问题 + AI 预回答，演讲者补充 |

## 三、扫码问卷功能

| 功能 | 说明 |
|:-----|------|
| 开场扫码 | 观众扫码进入问卷页面 |
| 填写 | 描述 AI 使用场景或遇到的问题 |
| 分类 | AI 自动识别为"分享"（使用场景）或"提问"（遇到问题） |
| 匹配 | 自动关联到 PPT 对应页面 |
