# ⚠️ 必须遵守的要求清单（每次改代码前先读一遍）

## 一、控制台（control.html）改法

### 1.1 布局结构
```
┌──────────────────────────────────┐
│ 🎙 当前页/总数   ● 已连接        │ ← 固定置顶，不随滚动移动
├──────────────────────────────────┤
│ P1 P2 P3 P4 … P20 附             │ ← 固定置顶
├──────────────────────────────────┤
│ [▶ 展开答案]（只有Slide2/4才显示）│ ← 操作按钮
├──────────────────────────────────┤
│ 演讲稿内容（可滚动）              │ ← 只有这部分滚动
│ ...                              │
│ ...                              │
├──────────────────────────────────┤
│ ⏭ 下一页预览                     │
├──────────────────────────────────┤
│        ← 上一页 | 下一页 →       │ ← 固定底部
└──────────────────────────────────┘
```

### 1.2 固定/置顶要求
- 页码栏 + 页码网格：position: sticky; top:0; z-index高
- 底部翻页按钮：position: fixed; bottom
- 只有演讲稿区域可以滚动

### 1.3 操作按钮
- Slide 2：显示"▶ 展开答案"按钮，点击展开/收起答案卡片
- Slide 4：显示"▶ 展开三阶段"按钮，点击后分三步展开
  - 第一步：展开卡片1（预训练）
  - 第二步：展开卡片2（指令微调）
  - 第三步：展开卡片3（RLHF+舔狗）
  - 再次点击：全部收起
- 必须发送 `{slide: N, expand: true/false, card: 1/2/3}` 到API
- 必须在展示页接收端正确处理这些指令

### 1.4 页码
- Slide 1~20：显示 P1~P20
- 附录页：显示"附"
- 总页数显示20（不含附录）
- 第5页是【转折：四大工程总览】不是模型对比表
- 模型对比表放到附录（第21页）

## 二、展示页（display.html）改法

### 2.1 字体
- 所有字体放大一倍
- h1标题、正文、卡片文字全部加大

### 2.2 Slide 4 — 模型页交互
- 右边三阶段卡片：**互斥展开**
  - 控制台分三步发送指令
  - 第一步：card=1 → 展开预训练卡片，其他收起
  - 第二步：card=2 → 展开指令微调卡片，预训练收起
  - 第三步：card=3 → 展开RLHF卡片，指令微调收起
  - expand=false → 全部收起
- 局限案例：`mode="limitations"` → 展示三个局限（幻觉/知识截止/无真实理解）

### 2.3 Slide 2 — 外星人语言
- expand=true → 展示答案↓
- expand=false → 收起答案（恢复猜题状态）

### 2.4 页面内容
- 所有页面必须用真实内容，不要placeholder
- 对照演讲稿和分享会ppt大纲.md来写

## 三、之前犯的错误（不要再犯）

1. ❌ 写代码前没读大纲记录 → ✅ 每次先读大纲记录.md再动手
2. ❌ 凭记忆写内容 → ✅ 从演讲稿复制完整内容
3. ❌ 用placeholder代替真实内容 → ✅ 全部用真实内容
4. ❌ 控制台顶部页码跟着滚动 → ✅ 固定置顶
5. ❌ 卡片展开用一个按钮全展开 → ✅ 互斥展开，分步控制
6. ❌ 字体没有实际放大 → ✅ 检查是否真的生效
7. ❌ Slide编号错乱 → ✅ 核对大纲记录的结构
8. ❌ API交互没调试 → ✅ 控制台发指令后检查展示页是否收到
